简单认识原型及原型链-饥人谷前端路程

发表于 2018-07-05

原型
在 JavaScript 中，每当定义一个对象（函数也是对象）时候，对象中都会包含一些预定义的属性。其中每个函数对象都有一个prototype 属性，这个属性指向函数的原型对象。
列-四大函数对象：
Number函数内原型指的是Number.prototype属性
String函数内原型指的是String.prototype属性
Boolean函数内原型指的是Boolean.prototype属性
Object函数内原型指的是的Object.prototype属性

这些属性都是window下已经定义好的全局函数，只要你打开浏览器，它就已经在分配好各自的内存，也就是可以直接访问，为了方便其他对象可以通过此进行属性继承。

原型链：
JS 在创建对象（不论是普通对象还是函数对象）的时候，都有一个叫做proto 的内置属性，用于指向创建它的构造函数的原型对象。
由于proto是任何对象都有的属性，所以会形成一条proto连起来的链条，递归访问proto直到尽头，尽头值为null。
当JS引擎查找对象的属性时，先查找对象本身是否存在该属性，如果不存在，会在原型链上查找，但不会查找自身的prototype
