HTML简介-饥人谷前端路程

发表于 2018-06-06

W3C简介：
了解前端起源的过程中，你会经常碰到TimBerners-Lee的名字，这个人是谁，他以Web之父及大多数浏览器标准的监护人而闻名于世。而WWW协会(WorldWideWebConsortium，也称W3C)的组织。它是由TimBerners-Lee于1994年创建的。W3C提出的目标是制定所有Web开发人员均可遵循的标准（如HTML这一控制Web页面外观的语言）。现在主流浏览器都遵循其指定的协议。

什么是W3C？
该组织在其Web站点上宣称：WWW协会的存在是为了发挥Web的全部潜力。换句话说，它的目标就是开发每个人都能够使用的WWW规格。该组织不致力于下层网络协议，比如一台机器如何与另一台机器进行通信。相反，它将重点放在人们在Web上冲浪时所看到的东西，如字体、图形和3D动画。该组织还就一些社会性的问题，如安全、知识产权、隐私访问等提出建议。

总结后有以下几点：
W3C 指万维网联盟（World Wide Web Consortium）
W3C 创建于1994年10月
W3C 由 Tim Berners-Lee 创建
W3C 是一个会员组织
W3C 的工作是对 web 进行标准化
W3C 创建并维护 WWW 标准
W3C 标准被称为 W3C 推荐（W3C 规范）

MDN简介：
全称Mozilla Developer Network谋智（火狐开源项目）开发者网络工作，里面有所有的互联网开发参考，
Mozilla 开发者网络（MDN）提供有关开放网络技术（Open Web）的信息，包括 HTML、CSS 和万维网及 HTML5 应用的 API，Mozilla 产品的文档，例如 Firefox OS。
Mozilla开发者网络（MDN）是一个完整的学习平台，你可以在这里深入学习网络技术以及能够驱动网络的软件，
MDN的使命很简单，就是要为一切开源的东西提供完整的、准确的、有用的文档，不管它是不是被火狐内置的软件所支持。
在里边你能清楚而准确的查询到自己想要的信息，在学习过程中遇到不太了解的前端知识都可以在这里边搜索。
更重要的Mozilla的社区遍及全球！Mozilla拥有来自世界各地、使用不同语言的了不起的贡献者。

各标签对应的英文全称，这样你便可以通过全称来更快的记住这些标签的具体含义：

其中常用到的有：
p&br&hr
p 表示段落，默认段落上下会有行间距；
br 是换行，自闭和标签；
hr 是横线；
a标签
1、可添加超链接标签；
2、有锚的作业，相当于业内定位；
H标签-标题标签
h1 字体号最大
h2 字体大小随着“h”后面的数字变大，而变小
h3 当然这个大小并非不可调试
h4
h5
h6 在原始标签中6号为最小号;
select标签
即我们常见的下拉框的标签;
Input 系列
标签用于搜集用户信息。
根据不同的 type 属性值，输入字段拥有很多种形式。输入字段可以是文本字段、复选框、掩码后的文本控件、单选按钮、按钮等等；
table：表格 li:列表；
iframe标签
用来在页面的一个框体中展示url的网页内容；

在学习标签的过程中还要了解以下几点：
空元素 ( 在 HTML标签中，没有内容的 HTML 标签被称为空标签 )

br //换行

hr //分隔线

input //文本框等

img //图片

textarea //标签定义多行的文本输入控件

select //元素可创建单选或多选菜单

link //标签最常见的用途是链接样式表

meta //元素可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词

空元素也叫可替换元素，就是浏览器根据元素的标签和属性，来决定元素的具体显示内容。
